# YouTube API Setup

## Quick Start

1. **Get API Key**: Go to https://console.cloud.google.com/
2. **Enable YouTube Data API v3**
3. **Create API Key**
4. **Add to .env**: `EXPO_PUBLIC_YOUTUBE_API_KEY=your_key_here`
5. **Restart app**
